## Comment 


* You can add comments to your HTML source by using the following syntax:

```HTML
<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Comments are not displayed in the browser -->

or

<!-- <div class="card">
        <div class="card-body">
            <ul class="list-group"> -->
```
